# 52Hz Sound Cabinet

Minimal Next.js 14 website scaffold for a sound art research initiative.

## Start

```bash
npm install
npm run dev
```

## Routes

- /
- /about
- /projects
- /projects/[slug]
- /research
- /immersive-performances
- /field-expeditions
- /press
- /publications
- /lectures
- /media-kit
- /contact

## Notes

- Uses semantic HTML sections with `h1` and `h2` hierarchy.
- Includes page-level metadata for basic SEO.
- Includes `StructuredDataPlaceholder` component in layout to prepare for future Schema.org JSON-LD insertion.
